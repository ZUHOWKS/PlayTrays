# PlayTrays

L'objectif est de développer une application de jeux sur plateau, ergonomique, moderne et ouverte à tout type
de public.

## Crédits

avec l'**Université Savoie Mont Blanc** et avec l'encadrement de **Flavien VERNIER**.
